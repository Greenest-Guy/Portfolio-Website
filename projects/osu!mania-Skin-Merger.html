About This Project

Published open-source software developed using Python with an intuitive GUI to merge and organize players’ UI configuration files for osu!mania, a rhythm game with a sparsely documented skinning system.

Garnered over 400+ downloads and received extensive positive feedback from the rhythm game community.

Software underwent multiple rounds of testing by experts in the rhythm game community, with later versions being published based on their feedback.
